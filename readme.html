Generating New Mavlink Source from automod.xml

 

1.     Clone the repo to somewhere reasonable:

 

$ cd ~/dev

$ git clone https://github.com/automodality/mavlink mavlink

 

2.     Run the generation script:

 

$ cd mavlink

$ python mavgenerate.py

 

3.     Enter the following in the dialog:

á      XML: message_definitions/audomod.xml

á      Out: AutoModUtils project src directory for output

á      Language: C

á      Protocol: 1.0

á      Validate: unchecked
